# 一个web项目demo
使用[warp](https://docs.rs/warp/latest/warp/index.html)、[rbatis](https://github.com/rbatis/rbatis)等常用库。
```
warp = "0.3.3"
rbatis = { version = "4.0"}
rbdc-mysql={version="0.1"}

rbs = { version = "0.1"}
tokio = { version = "1", features = ["full"] }
serde = { version = "1", features = ["derive"] }
```
